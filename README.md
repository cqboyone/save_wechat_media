# save_wechat_media
自动保存指定微信群的照片到指定位置/NAS

## 功能 
- [x] 将指定的群组中发的照片保存到本地
- [ ] 将指定的群组中发的照片保存到NAS

## 使用方法
1. 在[smallevilbeast/ntchat](https://github.com/smallevilbeast/ntchat)下载指定版本的PC微信客户端
2. 配置conf/config.yaml
3. 运行main.py
4. 打不来包，所以只有本地跑
